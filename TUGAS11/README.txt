# DASHBOARD HARGA PERTANIAN

## CARA MENJALANKAN:

1. Install library:
   pip install -r requirements.txt

2. Pastikan file 'harga_pertanian.csv' ada di folder yang sama

3. Jalankan aplikasi:
   streamlit run app.py

4. Buka browser di http://localhost:8501

## STRUKTUR FILE:

app.py          -> File utama dashboard
utils.py        -> Fungsi helper
requirements.txt -> Library yang dibutuhkan
harga_pertanian.csv -> Data CSV

## FITUR:

1. Filter data (tahun & komoditas)
2. Grafik tren harga per bulan
3. Perbandingan harga per wilayah
4. Distribusi harga (boxplot)
5. Tampilan data detail
